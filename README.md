Video plan game

1. Install game - npm install

2. Build game - npm run build

3. Start game - npm run game

"Press q :  Move back",
"Press w :  Move top",
"Press a :  Move left",
"Press e :  Move bottom",
"Press s :  Shoot ",
"Press d :  Start start plan to move angle velocity direction , press d again stop plan to move angle velocity direction",
